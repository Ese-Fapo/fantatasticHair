# Fantastic Hair

A React + Vite website for a hair salon, built with Tailwind CSS and a WhatsApp floating contact widget.

## Project structure

- `vite-project/` — main frontend app
- `vite-project/src/` — React source code and components
- `vite-project/public/` — static assets and public files

## Features

- React 19 frontend
- Vite development environment
- Tailwind CSS styling
- Floating WhatsApp contact component
- SEO-friendly static assets

## Quick start

1. Open a terminal in the workspace root.
2. Change into the app folder:

   ```bash
   cd vite-project
   ```

3. Install dependencies:

   ```bash
   npm install
   ```

4. Start the development server:

   ```bash
   npm run dev
   ```

5. Open the local URL shown in the terminal.

## Available scripts

From `vite-project`:

- `npm run dev` — start Vite development server
- `npm run build` — build production assets
- `npm run preview` — preview the production build
- `npm run lint` — run ESLint checks

## Notes

This repository contains the frontend app inside `vite-project`. If you want to modify the site layout or styling, start with `vite-project/src/App.jsx` and the components under `vite-project/src/components`.
